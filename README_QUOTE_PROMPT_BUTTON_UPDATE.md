# NestHelper Quote Prompt Button Update

Small admin dashboard update.

## Changed

- Added a visible **Quote helper** panel inside service request details.
- Added **Copy quote prompt** button.
- The button copies a ChatGPT-ready quote prompt using the selected request details.
- Prompt includes service guidance, pricing guidance, contact/service/scheduling details, uploaded-photo count, Smart Label pricing, and customer notes.
- No OpenAI API key needed.
- No auto-quote and no auto-send behavior added.
- No card, services page, header, footer, branding, or public page layout changes.

## Notes

This is intentionally the safer first version. It saves copy/paste time while keeping quote decisions reviewed manually.
