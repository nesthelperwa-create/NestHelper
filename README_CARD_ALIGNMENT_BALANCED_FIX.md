# NestHelper card alignment balanced fix

Small CSS-only card fix. Restores visually aligned closed service cards without forcing all cards in a row to stretch to the tallest open/expanded card.

Changed:
- Service card titles reserve a consistent two-line area when closed.
- Price/helper panel reserves a consistent height so buttons line up better.
- Cards still keep natural height when details are opened, so one expanded card does not force the rest of the row to become huge.
