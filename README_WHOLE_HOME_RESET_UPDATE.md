# Whole Home Reset update

This replacement update adds a separate `Whole Home Reset` option for regular full-home cleaning without changing the visual styling system of the request form.

## Main changes

- Adds `Whole Home Reset` as a service option.
- Keeps the existing card/form styling and two-column field layout.
- Adds a conditional Whole Home request section with:
  - home type
  - square footage
  - bedrooms
  - bathrooms
  - whole-home cleaning type
  - condition level
  - product preference
  - recurring interest
  - optional detail add-ons
  - top priorities/safety notes
- Keeps `Specific Area(s) Reset` for selected rooms/areas only.
- Keeps room-specific add-ons for Specific Area(s) Reset.
- Hides Smart Label setup from the Whole Home Reset flow so regular cleaning customers are not overwhelmed.
- Updates admin details, confirmation emails, request validation, referral handling, and family quote presets.

## Customer routing

Use `Whole Home Reset` when the customer wants regular cleaning for the entire home.

Use `Specific Area(s) Reset` when the customer wants selected rooms or zones only, such as kitchen + bathrooms, garage, pantry, closet, playroom, or laundry room.
