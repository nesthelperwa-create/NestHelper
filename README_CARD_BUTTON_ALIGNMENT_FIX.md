# Card Button Alignment Fix

Small ServiceCard-only patch.

## What changed

- Keeps service cards at a consistent closed height on desktop.
- Uses fixed closed title/description space so card content starts at the same vertical positions.
- Uses a fixed closed pricing panel height so shorter cards do not pull the buttons upward.
- Keeps expanded cards natural height so opening one card does not stretch the rest of the row.
- No branding, color, header, footer, button style, or page redesign changes.
